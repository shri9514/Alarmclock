# Alarmclock
Digital Alarm Clock
A basic Alarm Clock
Link : https://github.com/shri9514/Alarmclock.git

Features
Set : Set the Alarm
Delete : Delete the Alarm
Display : Shows the Alarm List

Data (in code)
'hr' : keeps track of hours
'min' : keeps track of minutes
'sec' : keeps track of seconds
